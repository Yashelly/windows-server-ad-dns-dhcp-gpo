# 0001 - Scope and non-goals

## Scope
- Build a small Windows Server lab that demonstrates practical infrastructure setup:
  - AD DS (new forest)
  - DNS
  - DHCP
  - Group Policy (example hardening policy)
- Provide clear, repeatable steps and validation commands.
- Keep evidence organized and easy to review.

## Non-goals
- Production-ready hardening baseline for an enterprise (this is a portfolio lab).
- HA/DR architecture (multi-DC, multi-site, clustering).
- Full security program (PKI, tiering model, PAM, SIEM, etc.).
