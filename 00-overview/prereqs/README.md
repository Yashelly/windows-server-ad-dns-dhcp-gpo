# Prereqs

## Lab assumptions
- Hypervisor: Hyper-V / VMware / VirtualBox
- Two VMs:
  - DC01 (Windows Server)
  - CL01 (Windows client)

## Networking
- Static IP for DC01
- DHCP scope for client subnet (example: 10.10.10.0/24)

## Tools
- Server Manager, AD DS, DNS, DHCP
- Group Policy Management (GPMC)
